# CosaNostra — клиент для браузера

Клиентская часть мессенджера CosaNostra: подключается к серверу по WebSocket и поддерживает создание пользователей и обмен сообщениями.

## Запуск

```bash
cd web
npm install
npm run dev
```

Откройте в браузере адрес, который выведет Vite (обычно http://localhost:5173).

## Сборка

```bash
npm run build
```

Результат в папке `dist/`. Для предпросмотра: `npm run preview`.

## Использование

1. **Подключение** — введите токен (в MVP это UUID пользователя), нажмите «Подключиться». Сервер должен быть запущен на `ws://localhost:8080/chat`.
2. **Создание пользователя** — после подключения заполните форму и нажмите «Создать». Валидация: имя 3–50 символов, пароль минимум 6 символов.
3. **Чат** — введите UUID чата (пока создание чатов на сервере не реализовано, для тестов используйте известный ID). Вводите сообщения и отправляйте. Поддерживается Enter для отправки.

Токен и ID чата сохраняются в `localStorage`.

## Реализовано по CLIENT_DEVELOPMENT.md

- Подключение по WebSocket с токеном (UUID)
- Ответ на ping (pong)
- Создание пользователя через WebSocket
- Отправка и получение сообщений с `clientMessageId` (дедупликация)
- Очередь сообщений при отключении и повторная отправка при восстановлении соединения
- Таймаут 5 с без ACK с уведомлением об ошибке
- Автоматическое переподключение при разрыве (до 10 попыток с интервалом 2 с)
- Локальное сохранение токена и chatId
- Отображение статуса сообщений (отправляется / отправлено / ошибка)
